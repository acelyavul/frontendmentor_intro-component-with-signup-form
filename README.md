## Installation


1. Clone the repo

```sh
https://github.com/acelyavul/frontendmentor_intro-component-with-signup-form.git
```

2. Install all the dependencies inside the project folder

```sh
yarn
```

3. Start the project

```sh
yarn start 
```

4. Run test

```sh
yarn test 
```
